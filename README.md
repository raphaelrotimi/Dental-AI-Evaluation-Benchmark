# Dental-AI-Evaluation-Benchmark

A professional benchmark for evaluating Large Language Models (LLMs) on dental healthcare tasks using structured human evaluation.

---

# Project Overview

This repository contains a collection of manually curated AI evaluation tasks focused on dental healthcare. Each dataset entry compares multiple AI-generated responses using a standardized human evaluation framework.

The benchmark assesses:

- Instruction Following
- Factual Accuracy
- Clinical Safety
- Response Completeness
- Response Clarity

The goal is to demonstrate practical experience in AI response evaluation, response ranking, rubric-based assessment, and benchmark dataset development.

---

# Benchmark Goals

This benchmark is designed to evaluate whether Large Language Models can:

- Follow user instructions accurately
- Provide factually correct dental information
- Maintain clinical safety
- Avoid unsupported diagnoses
- Produce complete and helpful responses
- Communicate information clearly and professionally

---

# Target Models

This benchmark can be used to evaluate responses generated by models such as:
