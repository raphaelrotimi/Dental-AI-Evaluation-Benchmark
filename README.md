# Dental-AI-Evaluation-Benchmark

A professional benchmark for evaluating Large Language Models (LLMs) on dental healthcare tasks using structured human evaluation.

## Project Overview

This repository contains a collection of manually curated AI evaluation tasks focused on dental healthcare. Each dataset entry compares multiple AI-generated responses using a standardized evaluation rubric.

The benchmark assesses:

- Instruction Following
- Factual Accuracy
- Clinical Safety
- Completeness
- Clarity

The goal is to demonstrate practical experience in AI evaluation, response ranking, rubric-based assessment, and dataset creation.

## Evaluation Workflow

Each dataset entry contains:

1. Evaluation Metadata
2. Clinical Scenario
3. User Prompt
4. Candidate Responses
5. Response Ranking
6. Rubric Evaluation
7. Final Justification
