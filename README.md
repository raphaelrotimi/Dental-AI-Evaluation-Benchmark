## Project Overview

The **Dental AI Evaluation Benchmark (DAEB)** is a domain-specific benchmark designed to assess the quality of AI-generated responses to dental healthcare questions.

Each dataset entry evaluates multiple candidate responses using a standardized human evaluation framework that emphasizes:

- Instruction Following
- Clinical Safety
- Factual Accuracy
- Completeness
- Clarity

Rather than measuring model performance automatically, this benchmark focuses on **human preference evaluation**, making it suitable for AI evaluation, RLHF workflows, quality assurance, and benchmark development.

---

## Objectives

This benchmark was created to:

- Evaluate AI-generated dental responses using human judgment.
- Demonstrate structured AI response evaluation techniques.
- Promote clinically safe educational healthcare responses.
- Showcase rubric-based assessment methodologies.
- Build a reusable benchmark for LLM evaluation research.

## Evaluation Methodology

Each dataset entry follows the same evaluation workflow:

1. Clinical Scenario
2. User Prompt
3. Three Candidate AI Responses
4. Human Preference Ranking
5. Rubric-Based Evaluation
6. Final Evaluation Summary
7. Overall Winner
8. Key Evaluation Takeaways

---

## Evaluation Rubric

Each candidate response is evaluated using five core criteria:

| Criterion | Description |
|-----------|-------------|
| Accuracy | Medical and factual correctness |
| Instruction Following | Compliance with prompt instructions |
| Completeness | Coverage of important information |
| Clarity | Organization and readability |
| Safety | Clinical safety and risk assessment |
