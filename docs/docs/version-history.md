# Version 1.0.0

**Release Status:** Stable

**Release Date:** July 2026

---

## Overview

Version 1.0.0 is the first official release of the Dental AI Evaluation Benchmark (DAEB).

This release establishes a complete, standardized benchmark for evaluating Large Language Model (LLM) responses to dental healthcare prompts using structured human evaluation.

---

## What's Included

### Dataset

- 30 benchmark dataset entries
- 90 AI candidate responses
- Standardized evaluation template
- Human preference rankings
- Rubric-based assessments

---

### Evaluation Framework

Each dataset includes:

- Evaluation Metadata
- Clinical Scenario
- User Prompt
- Three Candidate Responses
- Response Ranking
- Rubric Evaluation
- Final Evaluation Summary
- Overall Winner
- Evaluation Confidence
- Key Evaluation Takeaways
- Skills Demonstrated
- Evaluation Tags

# Version Numbering

This project follows Semantic Versioning.

- Patch version: Bug fixes, corrections, or minor refinements.
